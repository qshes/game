King of Vally

ENGLISH

This is a 2D game prototype written in C# using Unity.

The project focuses on character movement, interaction with the environment, and simple gameplay mechanics. It is not a finished game.

The player controls a king character. Movement includes directional animations: the character changes orientation depending on the direction (front, back, left, right).

The game includes interactive elements:
- levers
- pressure plates
- doors
- traps
- enemies

Pressure plates can open doors when activated. They can be triggered either by the player standing on them or by throwing a cube onto them.

There is also a simple inventory system. The player can pick up cubes, store them, and throw them using a key.

Implemented features:
- directional movement with animations
- interaction between game objects
- throwable and pickable objects
- simple inventory logic
- traps that deal damage
- enemies that kill the player
- basic level logic with triggers

This project was mainly about understanding how different game systems interact with each other inside a level.

РУССКИЙ

Это 2D игровой прототип, написанный на C# с использованием Unity.

Проект сделан с упором на управление персонажем, взаимодействие с окружением и игровые механики. Это не законченная игра.

Игрок управляет персонажем-королём. Реализованы анимации движения по направлениям: персонаж меняет ориентацию в зависимости от направления движения (вверх, вниз, влево, вправо).

В игре есть интерактивные элементы:
- рычаги
- нажимные плиты
- двери
- ловушки
- враги

Плиты открывают двери, если на них встать или бросить на них кубик.

Также реализован простой инвентарь: игрок может подбирать кубики, хранить их и бросать по нажатию клавиши.

Реализовано:
- передвижение с анимациями
- взаимодействие объектов
- подбор и бросок предметов
- простая логика инвентаря
- ловушки с уроном
- враги
- базовая логика уровня
